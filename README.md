# DevSticky

A lightweight sticky notes app for developers with syntax highlighting, code snippets, and cloud sync.

## Features

- **Syntax Highlighting** - C#, JavaScript, JSON, SQL, XML, Python, Markdown, and more
- **Lazy Loading** - Notes content loaded on-demand for fast startup
- **Always on Top** - Keep notes visible while coding
- **Code Snippets** - Save and reuse code with placeholders
- **Markdown Preview** - Real-time split-view preview
- **Note Templates** - Meeting Notes, Code Review, Bug Report, TODO List
- **Note Linking** - Link notes with `[[note-title]]` syntax
- **Cloud Sync** - OneDrive/Google Drive with AES-256 encryption
- **Global Hotkeys** - `Ctrl+Shift+N` new note, `Ctrl+Shift+D` toggle all
- **Multi-Monitor** - Notes remember their monitor
- **Organization** - Folders, Tags, Smart Collections, Kanban, Timeline
- **Fuzzy Search** - Intelligent search with typo tolerance
- **Auto-Backup** - Automatic backup with restore

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+N` | New note (global) |
| `Ctrl+Shift+D` | Toggle all notes (global) |
| `Ctrl+Shift+Q` | Quick capture (global) |
| `Ctrl+Shift+F` | Format JSON/XML |
| `Ctrl+F` | Find in note |
